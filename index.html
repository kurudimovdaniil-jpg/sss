О проекте

Это простой и понятный чат-бот, который отвечает на сообщения по заранее заданным правилам. Проект подходит для индивидуальной работы: код короткий, логика прозрачная, размещение бесплатное.

  • Frontend: чистые HTML, CSS, JavaScript
  • Бот отвечает локально (без интернета и API)
  • Опционально: подключение OpenAI API (платный ключ)
  • Можно легко расширить сценарии диалога
  • Публикация через GitHub + Vercel
